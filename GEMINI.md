# Gemini Project Configuration

This file stores key configuration details and conventions for the "The Amy Project" to ensure consistent and efficient assistance.

## Project Structure

*   **`amy/`**: Main application source code.
    *   **`core/amy_agent/agent.py`**: Defines the primary `root_agent`.
    *   **`integrations/telegram/bot.py`**: Logic for the Telegram bot interface.
*   **`frontend/`**: Web frontend implementation (React/Grok Mimic).
*   **`scripts/`**: Utility scripts and runners.
    *   **`run_web.py`**: Configuration for the ADK web UI.
    *   **`run_amy_bot.py`**: Runner for the Telegram bot.
    *   **`debug/`**: Debugging tools and utilities.
    *   **`management/`**: Project management scripts.
    *   **`amy_memory.db`**: The primary SQLite database for conversation history.
*   **`venv/`**: Python virtual environment.
*   **`.env`**: Stores environment variables (API keys, tokens).
*   **`requirements.txt`**: Lists all Python dependencies.
*   **`start_telegram.sh`**: Launcher script for the Telegram bot.
*   **`start_web.sh`**: Launcher script for the ADK web UI.

## Key Files & Variables

*   **Agent Definition**: `amy/core/amy_agent/agent.py` contains the `root_agent`.
*   **Database Location**: The SQLite database is located at `instance/amy_memory.db`.
*   **Environment File**: The `.env` file is located in the project root.

## Agent Logging

All logs generated by agents should be stored in the `agent_logs/` directory at the project root.

### `agent_logger.py` Module

This module (`amy/core/agent_logger.py`) provides a generic logging setup for all agents.

```python
from amy.core.agent_logger import setup_agent_logger

logger = setup_agent_logger("YourAgentName")
logger.info("This is a log message from YourAgentName.")
```
